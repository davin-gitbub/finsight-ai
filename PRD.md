# FinSight AI — 产品需求文档 (PRD)

**版本：** v1.0  
**日期：** 2026-06-25  
**状态：** Draft

---

## 1. 产品概述和目标

### 1.1 产品背景

FinSight 证券公司需在官网部署一个 AI 智能助手，帮助客户快速获取基于公司知识库的准确信息。

### 1.2 产品定位

FinSight AI 是嵌入 FinSight 证券公司官网的智能金融助手，基于检索增强生成（RAG）技术，以专业、简洁的中文风格回答用户问题。

### 1.3 产品目标

- 提供准确的知识库问答（准确率 >= 90%）
- 提升官网用户自助服务率
- 首 token 响应时间 < 2s，完整回答 < 10s
- 每次回答附带来源引用

---

## 2. 目标用户

- **个人投资者**：查询个股、行业动态、金融术语
- **机构客户**：查阅研究报告、公开信息
- **潜在客户**：了解 FinSight 公司服务

---

## 3. MVP 功能清单

**P0（必须有）：**

- [ ] RAG 问答 — Chroma + sentence-transformers + Claude API
- [ ] WebSocket 流式输出
- [ ] 多轮对话上下文（最近 3 轮）
- [ ] 10 分钟超时自动清空
- [ ] 引用来源标注
- [ ] 用户反馈 👍/👎
- [ ] 身份声明与拒绝回答规则
- [ ] PDF 文档解析与入库
- [ ] ChatWidget React 前端组件

**P1（重要）：**

- 思考过程展示（检索中...分析中...）
- Markdown 渲染
- 关键词推荐追问

---

## 4. 用户流程

```
打开页面 → ChatWidget 浮窗 → 点击展开 → 欢迎语
  → 提问 → RAG 检索 → Claude 流式回答 → 显示来源 → 反馈按钮
  → 继续追问 / 10分钟超时自动清空
```

---

## 5. 边界限制

- ❌ 不提供投资建议、不预测股价
- ❌ 不做个性化推荐
- ❌ 不接入实时行情
- ✅ 仅回答基于知识库的问题
